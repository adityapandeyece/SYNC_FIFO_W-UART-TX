I recently designed an 8×8 Synchronous FIFO in Verilog HDL and behaviorally verified it using a comprehensive testbench.

The verification covered several functional and corner-case scenarios, including:

✅ Write operation when FIFO is Full

✅ Read operation when FIFO is Empty

✅ Simultaneous Read & Write at Full condition

✅ Simultaneous Read & Write at Empty condition

The FIFO was implemented using a circular buffer architecture with independent read/write pointers and status flag generation (full and empty). Functional verification through simulation waveforms confirmed the expected behavior under all test scenarios.

➕ To extend the project further, I integrated this FIFO with the UART Transmitter from my previous project (with minor modifications) and verified the complete design through simulation. The UART transmitter was configured for 9600 baud with a 50 MHz system clock.
